TL;DR Relocate does not collect, store, transmit, or sell any personal data. Everything stays on your device.

1. What Data We Collect

None. Relocate does not collect any personal information, browsing history, location data, or analytics of any kind.

2. How the Extension Works

  • Relocate overrides the browser's navigator.geolocation API to return coordinates you choose.
  • Your chosen coordinates are stored locally in chrome.storage.local on your device only.
  • No data is ever sent to any external server, API, or third party.

3. Network Requests

Relocate makes the following optional network requests:

  • Address Search (Nominatim): When you search for a location, the search query is sent to OpenStreetMap's Nominatim API. This is a public, free, and open-source geocoding service. No personal data is attached to these requests.
  • Update Check (GitHub): The extension periodically checks the GitHub Releases API to see if a newer version is available. Only the extension's version number is compared ‚Äî no personal data is transmitted.

4. Permissions Explained

  • storage: To save your preferences (coordinates, theme) locally.
  • scripting: To inject the geolocation override script into web pages.
  • activeTab / tabs: To communicate state changes to open tabs.
  • alarms: To schedule periodic update checks.
  • host_permissions (*://*/*): Required to override geolocation on all websites. No data from these sites is collected.

5. Third-Party Services

The only external services contacted are:

  • OpenStreetMap Nominatim ‚Äî for address search (API Docs)
  • GitHub Releases API ‚Äî for update notifications (API Docs)
  • OpenStreetMap tile server ‚Äî for the map display in the popup

6. Data Retention

All settings are stored in chrome.storage.local and are deleted when you uninstall the extension.

7. Children's Privacy

This extension does not knowingly collect data from anyone, including children under 13.

8. Changes to This Policy

If this policy changes, we will update the "Last updated" date above. Continued use of the extension constitutes acceptance of the updated policy.
